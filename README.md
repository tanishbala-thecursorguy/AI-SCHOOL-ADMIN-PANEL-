## 🚀 Features

- **Dashboard Analytics** - Real-time metrics and KPIs visualization
- **User Management** - Complete CRUD operations for user accounts
- **Course Management** - Administer educational content and modules
- **AI Control Panel** - Manage AI-powered features and settings
- **Anti-Cheat System** - Monitor and prevent fraudulent activities
- **PvP Arena** - Manage competitive gaming features
- **Tournament System** - Organize and track competitions
- **Leaderboards** - Real-time ranking and performance tracking
- **Wallet & Finance** - Payment processing and financial analytics
- **Notifications** - System-wide messaging and alerts
- **System Settings** - Configuration and customization options

## 🛠️ Tech Stack

- **Frontend**: React 18.3.1 with TypeScript
- **Styling**: Tailwind CSS with Radix UI Components
- **Build Tool**: Vite 6.3.5
- **UI Components**: Material-UI, Radix UI primitives
- **Charts**: Recharts for data visualization
- **Routing**: React Router 7.13.0
- **Form Handling**: React Hook Form
- **State Management**: Built-in React hooks

## 🚀 Getting Started

### Prerequisites

- Node.js 16+ installed
- npm or yarn package manager
- Git (for version control)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/tanishbala-thecursorguy/AI-SCHOOL-ADMIN-PANEL-.git
   cd AI-SCHOOL-ADMIN-PANEL-
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

4. **Open in browser**
   Navigate to `http://localhost:5173` (or the port shown in terminal)

### Build for Production

```bash
npm run build
# or
yarn build
# or
pnpm build
```

The build output will be in the `dist/` directory.
